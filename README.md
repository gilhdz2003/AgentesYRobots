# Agentes&Robots

Portal B2B de productos y soluciones de Inteligencia Artificial empresarial.

## Stack

Vite + React 19 + TypeScript + Tailwind CSS 4 + Motion

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Sitio estático. Subir el contenido de `dist/` al hosting.
